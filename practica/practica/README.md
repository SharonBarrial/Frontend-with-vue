# Setup

## Install dependencies (short version)
```
npm install
```

## Install dependencies (longer version)
### Install PrimeVue
```bash
npm install primevue
```
### Install PrimeIcons
```bash
npm install primeicons
```
### Install VueRouter
```bash
npm install vue-router
```

### Install Axios
```bash
npm install axios
```
